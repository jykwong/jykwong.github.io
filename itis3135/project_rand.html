Site Map


Page Design

  • Homepage
    • Main page
    • Direct users to other pages within the site
    • Audience: users
    • Content: Images (food items) and navigation links
    • Users will not be entering data
    • Will contain hyperlinks
    • Side scrolling images based on daily specials. Hyperlinks to navigate site
  • About us
    • About us
    • Description of restaurant
    • Audience: users
    • Content: Contact information, paragraph description of the restaurant, social media business page, google maps embedded for the location
    • Users will not be entering data
    • Page will not contain buttons. Google maps embedded will be a hyperlink
    • No actions on the page aside from hyperlinks
  • Menu
    • Menu Page
    • Display menu items (food & drinks)
    • Audience: users
    • Content: Organized lists of drinks, appetizers, soups & salads, noodles, hibachi, sushi, chef specials
    • Users will not be entering data
    • Page will contain dropdown menu and a button to download a pdf of the menu
    • Dropdown menu will maximize/minimized when clicked on
    • Curated food or drink images will be shown
  • Gallery
    • Gallery Page
    • Image gallery of menu items (food & drinks) and candid photos
    • Audience: users
    • Content: Images of every menu item
    • Users will not be entering data
    • Page will contain dropdown menu
    • Dropdown menu will maximize/minimize when clicked on
    • Usage of dropdown menu intended to lower load times on the page
  • Order
    • Order Submission
    • User's place food order
    • Audience: user
    • Content: Submission form, order cost calculation
    • Users will be entering data
    • Data fields require validation
    • Page will contain dropdown menu, text box, submit button
    • Users will enter name, phone, payment method, food selection (through dropdown menu), custom requests per order (text box), process order (submit button)
    • Order page might not be functional until information security can be ensured

Dynamic Functionality

Website will contain sliding image gallery which will state the name of the item on mouseover and will serve as a hyperlink when clicked on to navigate the user to the item on the menu page. Submit order page will calculate the total cost of the user's order. Navigation links for the website will be kept in a "Standard Horizontal Menu" -- example shown in #1 Gate 39 UI Spotlight
